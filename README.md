## Project Overview

This project automates the (DemoQA)[https://demoqa.com/] website using Playwright with TypeScript.
The automation framework follows the Page Object Model (POM) design pattern
and focuses on reusability.

Key Objectives:
- End-to-End automation of DemoQA modules(WIP)
- Reusable page components
- Zero hardcoded waits
- Clean folder architecture
## Tech Stack
- Playwright
- TypeScript
- Node.js
- Page Object Model (POM)
- JSON Test Data
## Setup & Installation

1. **Clone the repository**<br>
   git clone <repo-link\>
   
2. **Install dependencies** <br>
   npm install
3. **Install Playwright with all the necessary browsers** <br>
   npm init playwright@latest
4. **Install Yarn**  <br>
   yarn install
6. **Run tests** <br>
   yarn taohid or npm run taohid
